## Career Transition Planning Checklist

### Define Your Goal

- [ ] What role do you want? (engineer, PM, design, etc.)
- [ ] Why? (be specific: "I want to ship products" not "I want a new job")
- [ ] Timeline: How soon? (6 months? 1 year? 2 years?)
- [ ] Constraints: What matters? (salary, work-life balance, learning, impact)

### Financial Readiness

- [ ] Current savings: $___
- [ ] Monthly expenses: $___
- [ ] Runway: ___ months
- [ ] Income needed: $___
- [ ] Timeline compatible with runway?

### Skill Assessment

- [ ] Current skills inventory
- [ ] Skills needed for target role
- [ ] Gap analysis (what's missing?)
- [ ] How to close gaps (bootcamp? self-study? courses?)

### Credibility Building

- [ ] Portfolio projects to build? (yes/no)
- [ ] People to talk to? (network in target role)
- [ ] Mentors to find? (someone who made similar transition)
- [ ] Learning path (courses, books, projects)

### Execution Plan

- [ ] Month 1-2: Learn foundations
- [ ] Month 3-4: Build 1-2 projects
- [ ] Month 5-6: Network and get referrals
- [ ] Month 7-8: Job search
- [ ] Month 9+: New role

### Job Search Preparation

- [ ] Resume updated with relevant experience
- [ ] Portfolio/projects ready to show
- [ ] LinkedIn profile optimized
- [ ] Target companies identified (50 companies)
- [ ] Referrals lined up (talk to 10+ people)

### Success Metrics

- [ ] How many interviews by [date]?
- [ ] How many offers by [date]?
- [ ] Target salary by [date]?
